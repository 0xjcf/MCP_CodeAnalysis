# @mcp/core

Core functionality for the MCP Code Analysis platform.

## Features

- Core analysis interfaces and types
- Shared utilities and helpers
- Common data structures
- Base analysis implementations

## Installation

```bash
pnpm add @mcp/core
```

## Usage

```typescript
import { Analyzer, AnalysisResult } from "@mcp/core";

// Example usage will be added as features are implemented
```

## Development

```bash
# Install dependencies
pnpm install

# Build
pnpm build

# Run tests
pnpm test

# Lint
pnpm lint
```
